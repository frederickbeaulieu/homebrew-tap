# Frederickbeaulieu Tap

## How do I install these formulae?

`brew install frederickbeaulieu/tap/<formula>`

Or `brew tap frederickbeaulieu/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "frederickbeaulieu/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
